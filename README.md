## 📌 Project Overview

This project is a **Blinkit Clone** built using **HTML, CSS, and JavaScript**. It recreates the look and feel of Blinkit's online grocery platform with a responsive design, product sections, shopping cart functionality, search features, and smooth user interactions.

The goal of this project is to practice modern front-end development concepts while creating a visually appealing e-commerce experience.
